# packages/

Per ADR-009 §8.1. Shared Go libraries + generated artifacts.

| Path | Role |
|---|---|
| `db/migrations/` | Atlas-managed schema migrations. `0001_init.sql` carries the `lecrm_provisioner` role bootstrap and the `core.lecrm_provision_workspace(uuid)` SECURITY DEFINER function. |
| `db/queries/` | `sqlc`-input `.sql` query files; generated Go code lives next to them. |
| `crm-adapter/` | The `CrmAdapter` Go interface — the contract that both `apps/api` and `apps/mcp` implement. Lives here so both consumers share types and there is exactly one definition (ADR-009 §4.2). |
| `shared-types/` | OpenAPI 3.1 → TypeScript codegen output for `apps/web/`. Generated by `@hey-api/openapi-ts` from the OpenAPI document emitted by the Go API (ADR-009 TO RESOLVE-6). |
| `tools/` | Mage tasks: codegen, lint pipelines, smoke scripts. |
